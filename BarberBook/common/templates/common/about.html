About the BarberBook Project

Hello, my name is Boyan Todorov and I currently study Python Web Development at SoftUni.

This is my project for my Web Development Module final assessment


BarberBook is a web application that helps clients book an appointment for their favorite barbershop in just few clicks. The app connects customers with barbershops for an easy reservation process. The app allows 2 types of users that can register on the website: Client and Barbershop.


Client users can choose from variety of barbershops and book an appointment.


Barbershops on the other hand can register on the website and can manage their reservations.



The project is developed using the following technologies:

  • For Back-end: Python, Django Framework
  • For Front-end: Custom HTML, CSS and additional components built with Bootstrap, Javascript, jQuery, LeafLet Maps Library
  • For Database: PostgreSQL
